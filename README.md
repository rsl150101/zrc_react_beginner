# 🚀 React 기초 학습 프로젝트 (ZRC React Beginner)

## 🚀 시작하기

각 섹션 디렉토리로 이동하여 필요한 패키지를 설치한 후 개발 서버를 실행할 수 있습니다.

```bash
# 섹션 이동
cd section1_gugudan

# 의존성 설치
npm install

# 빌드 및 실행 (Webpack)
npx webpack
```
*참고: 각 섹션마다 `webpack.config.js` 설정이 되어 있어 `npx webpack` 명령어로 빌드된 파일을 확인할 수 있습니다.*
